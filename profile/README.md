## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un sistema ERP enfocado en la gestión de **ventas, inventario y compras** para pequeñas y medianas empresas. La aplicación permitirá administrar usuarios con distintos roles, controlar el stock de productos, registrar cotizaciones y órdenes, y realizar seguimiento al estado de cada operación.

El objetivo es centralizar la información empresarial en una plataforma web segura, organizada y fácil de usar, que apoye la toma de decisiones y reduzca errores en los procesos administrativos.
